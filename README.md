# Healthcare Claims Analytics – Medicare Geographic Variation

## Project Overview
This project analyzes county-level variation in standardized Medicare spending per capita from 2019–2023.

## Key Findings (Preliminary)
- Spending distribution is right-skewed with high-cost outliers.
- Risk score and inpatient utilization show strong positive correlation with spending.
- Dual eligibility and readmissions also contribute meaningfully.

## Tools Used
- Python (Pandas, Seaborn, Matplotlib)
- Jupyter Notebook
- Git/GitHub
